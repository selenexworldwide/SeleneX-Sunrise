# SeleneX Careers Page Design

## Overview

A sophisticated, minimalist careers page that reflects SeleneX's mission of early cancer detection and precision care. The design emphasizes clinical elegance, international collaboration, and purpose-driven work.

---

### Visual Design
- **Background:** Full-width hero image with subtle overlay

  - Option B: Soft gradient from Deep Indigo (#1D1B4F) to Blue (#6377FF → #8AA8FF)
  - Overlay: 40% opacity white overlay for text readability
  
- **Animation:**

4. **Icons:**
   - Globe/Network (Remote work)
   - Clock (Flexible hours)
   - AI/Neural network (Technology)
   - Medical cross (Impact)
   - Book (Learning)
   - Shield (Ethics)
   - SVG format, 48x48px

### Animation Specifications

**Fade-in on Scroll:**
```css
@keyframes fadeInUp {
  from {
    opacity: 0;
    transform: translateY(30px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}
Duration: 0.6s
Easing: ease-out
```

**Card Hover:**
```css
transform: translateY(-4px);
box-shadow: 0 10px 25px rgba(0, 0, 0, 0.1);
transition: 0.3s ease-in-out
```

**DNA Helix Rotation:**
```css
@keyframes rotate {
  from { transform: rotate(0deg); }
  to { transform: rotate(360deg); }
}
Duration: 8s
Easing: linear
```

**Pulse Animation (Icons):**
```css
@keyframes pulse {
  0%, 100% { transform: scale(1); }
  50% { transform: scale(1.05); }
}
Duration: 2s
Easing: ease-in-out
```

## Responsive Design

### Breakpoints
- **Mobile:** < 640px
  - Single column layout
  - Stacked cards
  - Reduced font sizes (scale 0.9)
  
- **Tablet:** 640px - 1024px
  - 2-column grids
  - Adjusted spacing
  
- **Desktop:** > 1024px
  - Full 3-column layouts
  - Maximum content width: 1280px
  - Centered with padding

## Color Palette Usage

- **Background:** White (#FFFFFF) for main content
- **Text Primary:** Deep Indigo (#1D1B4F) for headlines
- **Text Secondary:** #666666 for body text
- **Accents:** Blue gradient (#6377FF → #8AA8FF) for CTAs and highlights
- **Borders:** Silver (#DBDBDB) for subtle dividers
- **Overlays:** White with 40% opacity for text readability on images

---

## Accessibility Considerations

- **Contrast Ratios:** All text meets WCAG AA standards (4.5:1 minimum)
- **Focus States:** 2px blue gradient outline on interactive elements
- **Keyboard Navigation:** All interactive elements accessible via keyboard
- **Screen Readers:** Proper ARIA labels and semantic HTML
- **Animation:** Respects `prefers-reduced-motion` media query

---

## Implementation Notes

1. **Performance:**
   - Lazy load images below the fold
   - Use WebP format with JPG fallback
   - Optimize animations (use `will-change` sparingly)

2. **SEO:**
   - Semantic HTML structure
   - Proper heading hierarchy (H1 → H2 → H3)
   - Meta descriptions for job listings

3. **Analytics:**
   - Track CTA clicks
   - Monitor scroll depth
   - Job listing views/interactions

4. **Integration:**
   - Job listings can be pulled from ATS (Greenhouse, Lever, etc.)
   - Contact form integration for "Get in Touch"
   - Newsletter signup for talent pipeline

---

## Content Guidelines

- **Tone:** Professional yet accessible, confident but humble
- **Focus:** Patient outcomes, privacy, ethical innovation
- **Avoid:** Overly technical jargon, unsubstantiated claims
- **Emphasize:** International collaboration, purpose-driven work, privacy-first approach
